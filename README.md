# Customer Retention & Churn Analysis

## Objective
Analyze customer churn and retention patterns using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Month-to-month contracts have highest churn
- High monthly charges increase churn probability
- Long-term customers are more retained
